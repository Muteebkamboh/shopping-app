shopping-app
A responsive e-commerce web application built using React.js, Redux Toolkit, and Tailwind CSS.

Features:
- Product Listing
- Add to Cart Functionality
- Redux Toolkit State Management
- Responsive UI
- API Integration
- Loading & Error Handling

Tech Stack:
- React.js
- Redux Toolkit
- JavaScript (ES6+)
- Tailwind CSS
- REST APIs
